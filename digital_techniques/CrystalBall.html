ReadMe File for Crystal Ball actions

Based on https://www.shutterfreaks.com/Actions/FreeStuff/CrystalBall/CrystalBall.html

How to install the action set

Open Photoshop.

If you haven't already done it, extract all the files from the ZIP file into a folder on your hard disk.

Click and drag the .atn file from that folder to your Photoshop window. If you look in the Actions window in Photoshop, you will see the action set appear there.

Open the action set by clicking the little arrow to the left of the action name. You will see one action inside. If you are currently connected to the internet, you will see a screenshot below showing the action set and action.


How to use the actions

  1. Open an image in Photoshop.

  2. Select the Crystal Ball action and click the "Play" arrow at the bottom of the Actions window. The action will make a copy of your image, leaving the original unchanged.

    Usually, you'll want to crop your image to a square. One easy way to do that is to hold down the Shift Key when you "draw" with the crop tool. Rectangular images can be fun too, though.


Notes
  1. The action will not work on images using some modes: Indexed Color and 16-bit, for example. If you see an error message similar to "The command Polar Coordinates is not currently available," change your photo to RGB, Lab, Grayscale, or CMYK mode and 8-bit mode before running the actions.
